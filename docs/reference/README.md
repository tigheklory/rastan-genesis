# Reference Artifacts

This directory holds durable validation/reference material that we want to keep
with the project.

Use it for:

- board capture notes
- extracted frame sets we still care about
- contact sheets and comparison images
- other evidence that supports reverse-engineering or validation

Subdirectories:

- [media](/home/tighe/projects/rastan-genesis/docs/reference/media) for extracted
  video frames, contact sheets, and recording reviews
- [images](/home/tighe/projects/rastan-genesis/docs/reference/images) for
  preserved comparison images and sprite probes

`build/` should be treated as disposable. If an artifact is worth keeping as
evidence, it belongs here instead.
